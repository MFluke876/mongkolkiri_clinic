After cloning:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev

# To use Ionic Capacitor:
# Step 1:
npm i -D @capacitor/cli

# Step 2:
npm run build

# Step 3:
npx cap sync

# Step 4: To open Android Studio
npx cap open android
# or
npx cap run android
# To run on native GUI
```
